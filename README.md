# Python Snake Game

Класична гра "Змійка", написана на Python з використанням бібліотеки Pygame.
Проєкт реалізовано з використанням об'єктно-орієнтованого підходу (OOP).

## Управління
- **Стрілки (⬆️⬇️⬅️➡️)**: Керування напрямком руху

## Особливості
- **Score System**: Лічильник очок у реальному часі.
- **Чиста архітектура**: Логіка розбита на класи (`classes_sn.py`) та ігровий цикл (`main.py`).

## Вимоги
- Python 3.x
- Бібліотека Pygame (`pip install pygame`)
